## Introduction
This is the first submission of `nysOpenData`.
The package provides a streamlined, catalog-driven interface for accessing NYS-related datasets from the NY State Open Data portal (https://data.ny.gov/), returning results as tidy tibbles with optional filtering, ordering, and date-based querying.

The goal of the package is to simplify access to public transit data and support reproducible research, teaching, and applied analysis using open data.

All checks were performed using devtools::check() and R CMD check --as-cran with no errors or warnings.
